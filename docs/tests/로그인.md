## 목적

사용자가 정상적으로 로그인 할 수 있어야 합니다.

## 대상

### `로그인 폼`

- **테스트 수준 및 유형**
  - 시각적 회귀 테스트 / E2E 테스트
- **시나리오**
  - 사용자가 아이디와 비밀번호를 입력할 수 있어야 함.
  - 입력된 비밀번호는 화면에 보이지 않아야 함.
- **환경 및 도구**
  - Cypress, Vitest
- **책임**
  - 전부

### `로그인 결과 확인`

- **테스트 수준 및 유형**
  - 시각적 회귀 테스트 / E2E 테스트
- **시나리오**
  - 로그인이 성공하면 로그인 된 화면으로 이동해야 함.
  - 로그인 실패 시 실패 메시지를 input 메시지 아래에 나타내야 함.
  - 로그인에 5회 실패 시 비밀번호 변경 알림을 주어야 함.
- **환경 및 도구**
  - Cypress, Vitest
- **책임**
  - 전부
